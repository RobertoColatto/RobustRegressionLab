# RobustRegressionLab

## Descrição do Projeto

Este repositório contém o RobustRegressionLab, uma aplicação web desenvolvida em Streamlit como parte do meu TCC. A plataforma tem como objetivo realizar análises de regressão em conjuntos de dados por meio de algoritmos clássicos e técnicas robustas, além de oferecer recursos para identificação de outliers e geração de insights sobre os dados.
